# 👤 User Flow

## Objetivo

Definir el recorrido que realizan los usuarios dentro de Puentes de Esperanza para completar una acción de forma rápida y sencilla.

---

# Flujo 1 - Quiero donar

Inicio

↓

Donaciones

↓

Elegir campaña

↓

Seleccionar método de pago

↓

Confirmación

↓

Gracias por tu ayuda ❤️

---

# Flujo 2 - Quiero donar insumos

Inicio

↓

Lista de insumos necesarios

↓

Encontrar centro de acopio

↓

Llevar la donación

↓

Donación recibida

# Flujo 3 - Quiero ser voluntario

Inicio

↓

# Flujo 4 - Busco un centro de acopio

↓

---

# Flujo 5 - Quiero informarme

Inicio

↓

Información verificada

↓

↓

Compartir información

---

# Principios de navegación

- Máximo 3 clics para llegar a cualquier función.
- Diseño simple e intuitivo.
- Información clara y verificada.
- Optimizado para dispositivos móviles.
- Accesible para cualquier usuario.
