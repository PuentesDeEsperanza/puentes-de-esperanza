
# 🌉 Puentes de Esperanza

> Uniendo personas. Multiplicando esperanza.

Puentes de Esperanza es una plataforma solidaria de código abierto que conecta personas, organizaciones y voluntarios para facilitar la ayuda humanitaria.

Nació del deseo de transformar la impotencia en acción, utilizando la tecnología para acercar a quienes necesitan ayuda con quienes desean brindar una mano.

## Nuestra misión

Facilitar la solidaridad mediante herramientas tecnológicas simples, accesibles y transparentes.

## ¿Cómo colaborar?

Muy pronto podrás colaborar como:

- 💻 Desarrollador
- 🧪 QA / Tester
- 🎨 Diseñador
- 🌍 Traductor
- ❤️ Voluntario
- 📦 Organizador de centros de acopio
